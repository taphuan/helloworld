* When capturing screen the user should have the possibility to record audio with the computer mic at the same time. * Whether to enable this feature or not should be available in the "Recording options"-dialog when initiating a screen recording in the same fashion as when you add voice in the current interface. * The label "Enable voice recording" and checkbox should be put in a separate row with the label to the left and the checkbox to the right. * The audio recording needs to be connected to the video in such a way that it could be disabled if the user wants to create an alternative voice over. It should also be possible to inform the user that the screen capture video file includes recorded audio.
